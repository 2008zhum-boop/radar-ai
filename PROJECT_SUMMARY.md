## 🎯 项目目标

1. **数据质检** - 识别和清除垃圾广告

## 🎓 使用场景

### 场景 1：清除垃圾广告
编辑发现大量垃圾广告混入采集数据：
1. 搜索"兼职"、"刷单"等关键词
2. 识别垃圾内容
3. 批量删除
4. 屏蔽垃圾信源

### 场景 2：发现行业热点
发生大事件（如 ChatGPT 发布新版本）：
1. 过滤"近1小时"的内容
2. 按热度排序
3. 看到该热点，虽未自动关联客户
4. 手动关联到"AI 科技组"

### 场景 3：纠正 AI 误判
AI 把"苹果发布会"误分为"农业新闻"：
1. 搜索"苹果"相关内容
2. 找到被误分的内容
3. 点击修正，改为"科技/新品发布"
4. 修改情感为"正面"

### 场景 4：维护黑名单
发现某营销号全是通稿：
1. 选中该信源的多条内容
2. 屏蔽信源，填写原因
3. 系统自动将该源加入黑名单
4. 之后该源的内容自动丢弃
